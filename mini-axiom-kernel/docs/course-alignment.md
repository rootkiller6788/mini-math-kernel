# Course Alignment ¡ª mini-axiom-kernel

Mapping of mini-axiom-kernel content to nine elite university curricula.

## MIT
- **18.701/702 Algebra**: Group theory axiomatization (Examples/GroupTheory.lean)
- **18.510 Mathematical Logic**: Axiom systems, soundness, completeness (Theorems/)
- **18.704 Logic II**: Compactness, deduction theorem (Theorems/Compactness.lean, Deduction.lean)

## Stanford
- **MATH 151 Logic**: Propositional logic encoding (Core/Objects.lean)
- **MATH 161 Model Theory**: Model finding and counting (Bridges/ToModel.lean)
- **MATH 290 Category Theory**: Categorical constructions (Constructions/Universal.lean)

## Princeton
- **MAT 520 Logic**: Axiom independence, completeness (Properties/Independence.lean)
- **MAT 560 Set Theory**: ZFC fragment encoding (Examples/SetTheory.lean)
- **MAT 563 Model Theory**: Conservative extensions, equivalence (Morphisms/Equivalence.lean)

## Berkeley
- **MATH 250A Algebra**: Group axioms as formal systems (Examples/GroupTheory.lean)
- **MATH 225A Model Theory**: Logical consequence, entailment (Properties/Decidability.lean)
- **MATH 254A Number Theory**: Peano arithmetic fragment (Examples/Peano.lean)

## Cambridge (Part III)
- **Logic and Set Theory**: Classical axiom systems (Core/Objects.lean)
- **Category Theory**: Morphisms between axiom systems (Morphisms/)
- **Model Theory**: Finite model theory, compactness (Theorems/Compactness.lean)

## Oxford
- **B1 Logic**: Deduction theorem, soundness (Theorems/Soundness.lean)
- **C2 Category Theory**: Universal constructions (Constructions/Universal.lean)
- **C3 Model Theory**: Consistency strength ordering (Properties/Consistency.lean)

## ETH Zurich
- **401-3001 Algebra I/II**: Formal axiom methods (Core/)
- **401-3462 Logic**: Completeness theorem proof (Theorems/CompletenessTheorem.lean)
- **401-3132 Number Theory**: PA axiomatization (Examples/Peano.lean)

## ENS Paris
- **Mathematical Logic**: Full meta-logical framework (Theorems/)
- **Algebra**: Group/field axiom systems (Examples/GroupTheory.lean)
- **Model Theory**: Finite model analysis (Advanced/FiniteModelTheory.lean)

## Tsinghua University
- **Abstract Algebra**: Axiomatic method foundations (Core/)
- **Mathematical Logic**: Soundness and completeness (Theorems/)
- **Set Theory**: ZFC finite models (Examples/SetTheory.lean)
